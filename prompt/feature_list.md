# TechStore Feature List

## User Features (15)
1.  **Register/Login**: Email/Password auth, JWT based.
2.  **Homepage**: Featured products, Hero slider, Categories.
3.  **Product Search & Filter**: Search by name, filter by category/price.
4.  **Product Detail**: Images, description, related products.
5.  **Cart Management**: Add to cart, update quantity, remove item.
6.  **Checkout**: Shipping address input, Payment method selection (COD/Online).
7.  **Order History**: List of past orders with status.
8.  **Order Tracking**: Detail view of a specific order.
9.  **Wishlist**: Save products for later.
10. **Profile Management**: Update name, phone, address.
11. **Change Password**: Secure password update.
12. **Product Reviews**: Star rating and text review.
13. **Related Products**: Suggestions on product detail page.
14. **Notifications**: System notifications (Order status).
15. **Contact/Support**: Contact form.

## Admin Features (10)
16. **Admin Login**: Separate login for admins.
17. **Dashboard**: Statistics (Revenue, Total Orders, New Users).
18. **User Management**: View users, Block/Unblock.
19. **Product Management**: Add, Edit, Delete products. Image upload.
20. **Category Management**: Create/Edit/Delete categories.
21. **Order Management**: View orders, Update status (Pending -> Shipped -> Delivered).
22. **Inventory Management**: Track stock levels.
23. **Review Management**: Moderate user reviews.
24. **Discount/Coupon**: Create promo codes.
25. **Site Settings**: Manage slider images, contact info.
