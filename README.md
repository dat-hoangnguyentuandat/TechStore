# TechStore

A full-stack e-commerce application built with React (Vite) and Node.js (Express + MongoDB).

## Structure

- `client/`: Frontend application (React + Vite)
- `server/`: Backend application (Node.js + Express)

## Setup

### Client
```bash
cd client
npm install
npm run dev
```

### Server
```bash
cd server
npm install
npm run server
```
